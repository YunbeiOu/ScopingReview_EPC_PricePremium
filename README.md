# The Price Premium of Residential Energy Performance Certificates: A Scoping Review of the European Literature

Repository for research data/code for the scoping review paper. 

## Searching and screening

This folding contains all the files and data in the systematic literature searching and screening process. The files are in order, showing a sequential, transparant process.

The whole process include: (1) search command, (2) database searching results, (3) screening (two stages), (4) screening sample check.
